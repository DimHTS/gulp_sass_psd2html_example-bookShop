- необходимо установить node.js

1) установить зависимости, для этого необходимо в командной строке перейти в папку проекта и выполнить следующую команду:
   npm install

2) запуск сервера, для этого необходимо запустить файл run.bat, при этом должна открыться страница в браузере:
   http://localhost:3000/, если этого не произошло, то необходимо в браузере открыть страницу http://localhost:3000/

3) редактирование файлов происходит в папке app

4) после того как все правки были сделаны, необходимо сгенерировать дистрибутив, для этого необходимо в командной строке 
   перейти в папку проекта и выполнить следующую команду:
   gulp build

5) файлы дистрибутива будут расположены в папке dist
